## Running with Docker

1. Set the MFA environment variables.

    ```bash
    export MFA_METHOD="email"
    export SENDGRID_API_KEY="your_sendgrid_api_key_here"
    export FROM_EMAIL="your_verified_sender_email_here"
    ```

    Example:

    ```bash
    export MFA_METHOD="email"
    export SENDGRID_API_KEY="no"
    export FROM_EMAIL="Lia03gomes@gmail.com"
    ```

    Note: If SendGrid is actually being used, `SENDGRID_API_KEY` must be a valid SendGrid API key and `FROM_EMAIL` should be a verified sender email.

2. Build the backend Docker image.

    ```bash
    docker compose build backend
    ```

3. Run the backend container.

    ```bash
    docker compose up backend
    ```

4. Open a second terminal and watch the audit log.

    ```bash
    tail -f data/audit_log.txt
    ```

5. In a third terminal, test the endpoints.

    Login:

    ```bash
    curl -X POST http://127.0.0.1:5000/login \
      -H "Content-Type: application/json" \
      -d '{"username":"alice","password":"password123"}'
    ```

    Verify MFA:

    ```bash
    curl -X POST http://127.0.0.1:5000/verify-mfa \
      -H "Content-Type: application/json" \
      -d '{"username":"alice","code":"123456"}'
    ```
